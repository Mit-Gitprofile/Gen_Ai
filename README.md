## 📖 Overview

**AI Interview Voice Bot** is an intelligent interview preparation tool that helps job seekers practice behavioral interviews using artificial intelligence. The bot generates personalized questions based on your profile, records your spoken answers, and provides instant feedback using the STAR method evaluation framework.

### 🎯 Purpose

- Help candidates prepare for technical job interviews
- Practice behavioral questions in a realistic setting
- Receive AI-powered feedback on communication and structure
- Build confidence through repeated practice sessions
- Identify areas for improvement with specific tips

---

## ✨ Key Features

| Feature | Description |
|---------|-------------|
| 🎯 **Personalized Questions** | AI generates 5 behavioral questions tailored to your profile |
| 🎤 **Voice Recording** | Record answers using your browser's microphone |
| 📝 **Speech Transcription** | Automatic conversion of speech to text |
| 🤖 **AI Feedback** | Instant evaluation using the STAR method |
| 📊 **Performance Scores** | Get scores out of 10 for each answer |
| 💡 **Improvement Tips** | Specific, actionable advice for better answers |
| 📈 **Summary Report** | Review all answers and overall performance |
| 🔐 **Privacy First** | Your API key stays in your browser; no data stored |
